# Bloc°6 - Projet Final

## Description

This repository contains various projects related to Automatic License Plate Recognition (ALPR) and Computer Vision, developed as part of the final project for the Fullstack WSL course at Jedha. The projects include ALPR Engine, Detection of Cars, Model YOLO8-n & OCR, Model YOLO from Scratch, and Presentation.

## Installation

To run the projects, follow these steps:

1. Install the required Python packages by running `pip install -r requirements.txt` in the respective project directories.
2. For Docker-based projects, build and run the Docker container using the provided `Dockerfile`.

## Usage

Each project has its own structure and usage instructions, located in the respective `README.md` files.
